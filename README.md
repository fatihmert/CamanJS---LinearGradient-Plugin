# CamanJS - LinearGradient-Plugin

Easy using;

```javascript
Caman.gradientLinear(pos,main,gmap);
```

| pos           | main          | gmap  |
| ------------- |:-------------:| -----:|
| left          | blend         | color |
| right         | opacity       |opacity|
| bottom        |               |       |
| top           |               |       |
